Events

  • HTML item clicked

  • Form is submitted .submit()
  • Change a value .change()
  • Form is submitted .select()
  • Image is loaded .load()

  • User changes option select
  • Checkbox checked/unchecked
  • Show item clicked (When a ul list item is clicked)
